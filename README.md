### Pokretanje aplikacije:
  - Uraditi clone repozitorija
  - Instalirati docker
  - Unutar foldera u kojem se nalazi projekat, unijeti sljedeće komande na CMD:
```
docker-compose build
```
```
docker-compose up
```
